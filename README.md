# Day 01 – Variables & Data Types

## Variables
- var → function/global scope, can reassign
- let → block scope, can reassign
- const → block scope, cannot reassign

## Data Types
### Primitive
- Number, String, Boolean, Null, Undefined, Symbol, BigInt

### Reference
- Object, Array, Function

## Key Learnings
- let & const are block scoped
- Reassigning const throws error
- Arrays & Objects are reference types
